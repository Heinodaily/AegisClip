## 常见问题
1. **如何使用全局快捷键?**
   按下 Option+Command+V 即可打开AegisClip

2. **如何清空历史记录?**
   点击窗口右上角的"清空历史"按钮

3. **如何使用多选模式?**
   点击窗口右上角的多选按钮进入多选模式

## 隐私政策
AegisClip尊重并保护用户隐私。所有剪贴板数据均存储在本地设备上，不会上传至云端或与第三方共享。
